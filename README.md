# Ask To Me!

## 용도
익명 질문함 입니다. 사용자는 익명 질문을 남기는 것이 가능하고 질문자는 응답 및 응담에 대한 표기가 가능한 웹페이지 입니다.
사용자가 입력한 질문에 대해 수정, 삭제가 가능하게끔 최소한의 정보로 회원가입 or OAuth기능을 지원합니다.
이메일 인증도, 어떠한 경우의 사용자 인증도 없으므로 익명성을 보장합니다.
